### Swiggy Sales Analysis Dashboard (Excel Project)

An interactive Microsoft Excel dashboard that analyzes Swiggy sales, customer ratings, food preferences, and regional performance across India.

# Project Title

Swiggy Sales Analysis Dashboard

# Brief One-Line Summary

An Excel-based dashboard that transforms raw Swiggy order data into actionable insights on sales, ratings, orders, and customer preferences.

# Overview

This project is a complete data analysis and dashboarding solution built entirely in Microsoft Excel. Using a large Swiggy dataset containing over 197,000 records, the dashboard provides insights into:

Total Sales, Average Ratings, Average Order Value, Total Orders, Sales Trends (Monthly, Daily, Weekly), Top Cities by Sales, State-wise Sales Distribution, Food Type Analysis (Veg vs Non-Veg)

The dashboard is interactive and includes slicers for filtering by:
Month, Category, Restaurant Name,

# Problem Statement

Swiggy generates a huge amount of order data daily. Manually analyzing this data to identify business trends is time-consuming and inefficient.

The goal of this project is to build an interactive dashboard in Excel that helps answer business questions such as:

Which cities generate the highest sales?
What are the monthly and weekly sales trends?
What is the average customer rating?
Are Veg or Non-Veg items more popular?
Which states contribute the most revenue?

# Dataset

Columns in Dataset: State, City, Order Date, Day, Quarter, Week, Restaurant Name, Location, Category, Dish Name, Food Type, Price (INR), Rating, Rating Count

# Dashboard KPIs	                   
Total Sales - ₹53.01M, Average Rating - 4.34, Average Order Value	- ₹268.51, Rating Count	- 5.59M, Total Orders - 197.43K

# Key Insights
Bengaluru is the highest revenue-generating city.
Veg food accounts for the majority of sales (67%).
Average customer rating is consistently high at 4.34.
Sales remain stable across most weeks.
Q2 recorded the highest sales performance.

# Results and Conclusion

The project demonstrates how Excel can be used to build a professional and fully interactive dashboard. It converts raw transactional data into meaningful insights that support strategic business decisions.
